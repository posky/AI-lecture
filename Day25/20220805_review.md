### 앙상블 기법
* Random Forest
    * 여러 개의 트리 모델을 만든다.
    * 각각의 트리가 독립적이다.
    * 조금씩 다른 데이터 세트, 조금씩 다른 변수
    * 회귀 - 여러개 모델이 예측한 값의 평균
    * 분류 - 여러개 모델이 예측한 값의 다수로 결정.
* Gradient Boosting
    * 여러 개의 트리 모델을 만든다.
    * 여러 개의 트리가 의존적이다.
    * XGBoost, LightGBM
* 랜덤 포레스트, 그래디언트 부스팅
    * 기본 요소: 트리
        * 지니계수, 엔트로피, mse 등의 기준으로 노드를 분할.
        * max_depth: 트리의 깊이
